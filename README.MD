## Node-ts-multiplication-app

CLI desarrollado en Node.js con TypeScript que genera tablas de multiplicar a partir de argumentos por línea de comandos.
Características:

Argumentos configurables via yargs (base, límite y visualización en consola)
Escritura de resultados en archivos .txt
Testing con Jest y ts-jest
Migración completa de JavaScript a TypeScript

Tecnologías: Node.js, TypeScript, Yargs, Jest

===========================================================================
Instrucciones para correr y ejecutar nuestro programa

 > 1. Instalar dep
 ````
 npm install
 ````

 > 2. Correr
 ````
 npm run dev
 ````
